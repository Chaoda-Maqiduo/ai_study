## 知识沉淀库 (Knowledge Base)

### 01 HTML 语义化
- **舞台理论**: `<body>` 是所有可见内容的容器。`<head>` 仅用于元数据（如 `<title>`）。
- **语义 vs 样式**: 标签名应描述“这是什么”（如 `<article>`），而非“长什么样”。
- **AI 友好**: 语义化标签是 AI 提取网页信息的关键接口。
- **关键标签**: `<header>` (组头), `<nav>` (导航), `<main>` (主体), `<aside>` (侧边栏), `<article>` (文章), `<time>` (时间)。
- **深度复习 (2025-12-19)**:
    - 明确了 `<title>` (Metadata) 与 `<h1>` (Content) 的区别。
    - 掌握了基于 `<main>` 和 `<section>` 构建文档大纲 (Document Outline) 的方法。
    - 理解了页面骨架与 CSS 布局的关系。

### 02 CSS 盒子与布局
- **快递盒模型**: Content (产品) -> Padding (气泡垫/内白) -> Border (纸箱) -> Margin (快递间距/外距)。
- **背景色表现**: Background-color 包含 Padding 区域，但不包含 Margin。
- **Flexbox 三板斧**:
    - `display: flex`: 开启弹性布局。
    - `justify-content`: 管水平方向 (space-between, space-around, center)。
    - `align-items`: 管垂直方向 (center, stretch)。
- **Tailwind 原子类**: 习惯了 `flex`, `p-x`, `m-y`, `items-center` 等缩写逻辑。
- **交互态**: 掌握了 `hover:` 前缀实现鼠标悬停效果。
